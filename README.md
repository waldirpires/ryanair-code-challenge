# ryanair-code-challenge
RyanAir Code Challenge

This is a SpringBoot-based Java Application built as a AWS Lambda function to retrieve flight connections based on request parameters.

Steps:

1. Create the application + tests: OK
2. Create the AWS Lambda endpoint + tests
3. Create the entry point of the AWS Lambda in the cloud + configure API gateways
4. Associate entry point to the application deployed + tests

Next steps:
a. Add caching capability for Routes and Schedules API endpoints
